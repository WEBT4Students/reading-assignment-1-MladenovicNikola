World Wide Web General

What are the most essential technologies that made the World Wide Web such a success?

HTTPS (Hypertext Transfer Protocol), URL (Uniform Resource Locator), HTML, SMTP (Simple Mail Transfer Protocol), FTP (File Transfer Protocol).

How are resources (html pages, images, etc) addressed in the web?

They are adressed with the URL (Uniform Resource Locator).

What are the general components of a URL?

There are 5 components => Scheme (https://), Subdomain (www), Second-level-Domain (goggle), Top-level-Domain (com), Subdirectory (/ index.html/).

Web Server

What is a web server?

It is computer software and underlying hardware, that accepts requests via HTTPS or its secure variant HTTPS.

What does a web server?

It uses HTTP and other protocols (SMTP, FTP) to respond to client requests made over the World Wide Web (WWW).

The Main Job of a web server is to display website content through storing, processing and delievering webpages to users.

How can content be stored on a web server?

Thex store data on a server using some kind of database (server-side-storage). Then run server-side code to retrieve needed data, insert

it into static page templates, and serve the resulting HTML to the client to be displayed by the users browser.

How is content in webpages described?

Page content refers to all the information contained in a website. Page content can be displayed as text, links, images, audio, animation

or videos among other things.

Web Client

What is a web client?

It is an application used for connecting to a web server over HTTP (web browser Firefox).

What does a web client?

It retrieves it's files from a web server to find the a website which suits the user request and to graphically render the page on the users srceen.

Web Client and Web Browser are they the same?

Yes, they are the same.
